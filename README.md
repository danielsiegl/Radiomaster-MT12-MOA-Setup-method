# Radiomaster-MT12-MOA-Setup-method
This setup method will configure your Radiomaster MT12 remote control to be suitable for MOA crawlers. Also, set the S2 knob to DIG proportional adjustment.

### 1，Enter the input page and configure the channel and knob mapping.
![Project Screenshot](image/IMG20251202154728.jpg)

### 2，Enter the curve settings page, as shown in the image. Add two sets of 3-point curves.
![Project Screenshot](image/IMG20251202154809.jpg)
![Project Screenshot](image/IMG20251202154930.jpg)

### 3，Go to the MIXES page, as shown in the image, and add the corresponding mixing control entry.
![Project Screenshot](image/IMG20251202155030.jpg)
![Project Screenshot](image/IMG20251202155059.jpg)
![Project Screenshot](image/IMG20251202155128.jpg)
![Project Screenshot](image/IMG20251202155157.jpg)
![Project Screenshot](image/IMG20251202155212.jpg)

### 4，At this point, the setting for the ratio of the front and rear wheel rotations controlled by the S2 knob is complete. You can also add a midpoint indicator sound for the S2 knob to make it easier to know the midpoint position of the S2 knob during operation.
##### Go to page 8/12, as shown in the image, and add the corresponding parameters for L01.
![Project Screenshot](image/IMG20251202155404.jpg)
![Project Screenshot](image/IMG20251202155455.jpg)

##### Go to page 9/12, as shown in the image. Add the corresponding parameters for L01.
![Project Screenshot](image/IMG20251202155520.jpg)

### At this point, your S2 knob will play a corresponding beep when it returns to the center position.
